# 挂载 SMB 共享

SMB（Server Message Block，服务器消息块）是一种用于在网络上共享文件、打印机和其他资源的协议。如果你的局域网中有 NAS 设备或其他 SMB 服务器，可以轻松地将 SMB 共享挂载到 Olares 中来访问和管理共享文件。

## 准备工作

- 确保 Olares 与 SMB 服务器处于同一局域网中。
- 你已经获取以下信息：
    - SMB 共享路径，通常格式为 `//<IP地址>/<共享文件夹名>`。
    - 访问 SMB 共享的用户名和密码。

## 挂载 SMB 共享

1. 打开文件管理器，进入**存储** > **外部设备**。
2. 点击右上角的**连接服务器**。
3. 在弹出的对话框中，点击**协议**下拉菜单，然后选择 **SMB**。

    ![配置 SMB 连接](/images/zh/manual/olares/add-SMB-share-path.png#bordered)

4. 在**服务器地址**栏中，输入 SMB 共享路径。例如：`//192.168.1.100/Documents`。
5. （可选）保存常用服务器地址，方便下次快速连接：

    - 输入共享路径后，点击 <i class="material-symbols-outlined">add</i> 可将服务器地址添加到**常用服务器**。
    - 要删除已保存的地址，点击**常用服务器**下的地址，然后点击 <i class="material-symbols-outlined">remove</i>。

6. 点击**确认**。
7. 输入用户名和密码，然后点击**确认**。

    连接成功后，SMB 共享将显示在**外部设备**目录中，然后你就可以直接在此访问共享文件和文件夹了。

## 卸载 SMB 共享

不再需要访问网络文件时，你可以安全地断开共享。

1. 打开文件管理器，进入**存储** > **外部设备**。
2. 右键点击已挂载的文件夹，然后选择**卸载**。

    SMB 共享会立即与 Olares 断开连接，并从**外部设备**目录中被移除。
