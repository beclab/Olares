# Switch between Olares IDs

If you have added multiple Olares IDs in LarePass, you can switch between them on mobile app or desktop client.

## On mobile

1. Open the Switch account page in one of the following ways:

   - **From the main interface**: Tap your profile avatar in the top-left corner.

      ![Switch Olares ID from the main interface](/images/manual/larepass/switch-account-mobile1.png#bordered)

   - **During account creation**: Tap the account icon in the top-right corner.

      ![Switch Olares ID during account creation](/images/manual/larepass/switch-account-mobile2.png#bordered)

2. On the Switch account page, tap the account you want to use from the list.

   ![Select an account from the list](/images/manual/larepass/switch-account-mobile.png#bordered)

## On desktop

1. Click your profile avatar in the top-left corner, then click **Switch account**.
2. In the list, click the account you want to use.

   ![Switch Olares ID on desktop](/images/manual/larepass/switch-account-desktop.png#bordered)
