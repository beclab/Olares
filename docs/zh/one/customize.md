# 自定义 Olares <Badge text="15 min"/>

Olares 允许你自定义私人工作空间和公共身份。本指南将向你展示如何设置桌面外观，并设计一个可以与他人分享的个人主页。

## 学习目标

完成本教程后，你将学会：
- 在浅色和深色系统主题之间切换。
- 自定义桌面和登录屏幕壁纸。
- 使用社交链接和内容块设计你的公共个人主页。
- 为页面设置样式并分享它。

## 自定义 Olares 桌面

你可以通过系统设置调整本地 Olares 环境的外观和体验。

1. 从 Dock 打开 Settings。
2. 在侧边栏中选择 **Appearance**。

![自定义桌面](/images/one/setting-appearance.png#bordered){width=80%}

### 设置系统主题

在 **Theme** 中，选择 **Light** 或 **Dark** 模式。

更改会立即生效。

### 更换壁纸

你可以为桌面和登录屏幕设置不同的外观。

1. 在 **Wallpaper** 中，选择目标：**Desktop background** 或 **Login background**。
2. 要使用内置壁纸，从提供的图片库中浏览并选择。
3. 要使用自定义壁纸：

   a. 点击 **Upload image**，从电脑中选择图片文件，然后点击 **Open**。  
   b. （可选）使用 **Upload image** 旁边的下拉列表调整壁纸显示方式：  
      - **Stretch** 
      - **Fill**
      - **Tile**

#### 移除已上传的图片

已上传的壁纸会显示在单独的 **Uploaded images** 部分。

要删除已上传的图片，将鼠标悬停在其上，然后点击右上角的 <i class="material-symbols-outlined">close</i> 图标。

## 设计你的公共个人主页

你的 Olares 个人主页是一个公共首页，地址为 `https://{your-olares-id}.olares.com`。你可以将其用作落地页、作品集或数字名片。

![个人主页](/images/manual/olares/profile.png)

从 Dock 打开 Profile 开始编辑。编辑器有四个标签页：**Header**、**Social**、**Blocks** 和 **Appearance**。 

![Profile 图标](/images/one/profile-icon.png#bordered){width=95%}

:::tip 自动保存
所有更改都会在操作时自动保存。
:::

### 设置页头

使用 **Header** 标签页定义页面顶部的身份信息。你可以选择布局样式、上传头像，并设置显示名称。

![Profile 页头](/images/one/profile-header.png#bordered){width=95%}

:::tip 保持简洁
一个清晰的名字加上一行描述，通常足以给人留下良好的第一印象。
:::

### 添加社交链接

使用 **Social** 标签页在页头下方显示图标。这可以将访客连接到你在其他平台上的存在，如 X、GitHub 或 LinkedIn。

你可以添加多个平台的链接，并选择显示哪些。要暂时隐藏某个链接，将其关闭即可，无需删除已保存的值。

![Profile 社交链接](/images/one/profile-social.png#bordered){width=95%}

### 使用内容块构建页面

使用 **Blocks** 标签页构建页面的主体内容。点击 **+ Add Block** 并选择类型以进入编辑界面。

每种内容块类型帮助你分享不同的内容：
- **Text：** 添加标题或段落。你可以调整文本对齐方式和背景透明度。
- **Link：** 输入网站 URL 或电子邮件地址。你可以自定义按钮标题、副标题和阴影样式。
- **Image：** 上传照片。你可以添加描述并选择最适合图片的宽高比。

![Profile 内容块](/images/one/profile-add-block.png#bordered){width=95%}

添加内容块后，你可以在列表中直接管理布局：
- 拖动项目重新排序。
- 关闭某个内容块以暂时隐藏它。
- 复制某个内容块以复用其样式。

### 配置外观

使用 **Appearance** 标签页定义视觉风格。你可以选择一个预设主题，并手动微调字体和内容块形状以匹配你的风格。

![Profile 外观](/images/one/profile-appearance.png#bordered){width=95%}

## 分享你的个人主页

你的个人主页地址为 `https://{your-olares-id}.olares.com`。

准备好后，点击 **<i class="material-symbols-outlined">share</i> Share** 按钮。你可以：

- 直接分享到社交平台。
- 生成二维码。
- 导出一张可作为数字名片使用的可分享图片卡片。

![Profile 分享](/images/one/profile-share.png#bordered){width=95%}
