# 设置和管理 Olares

Olares 的**设置**提供了简洁直观的界面，让你轻松配置并管理 Olares。这里是个性化设置、系统安全、服务集成、应用管理等操作的中心入口。

设置包含以下功能模块：
- **我的 Olares**：查看账户资料，管理硬件、Olares Space、密码、网络访问策略，并查看 Olares OS 版本。
- **用户**：管理 Olares 系统内的用户账户。
- **外观**：设置语言、小组件、日期时间格式、仪表盘显示和主题。
- **应用**：配置应用入口、自定义应用域名，并管理应用环境变量。
- **集成**：集中管理外部连接，包括云存储、网站 Cookie 和 SMB 网络凭据。
- **VPN**：配置安全的远程访问，连接你的 Olares 设备。
- **网络**：通过更改反向代理、设置 hosts 文件，以及为支持的应用启用 Overlay Gateway 来管理网络路由。
- **AI 算力**：分配和监控包括 GPU 在内的加速器资源，用于硬件加速任务。
- **视频**：配置系统级的视频播放和转码偏好。
- **搜索**：设置索引规则，优化全局文件搜索体验。
- **备份**：创建和管理备份任务，保护你的系统数据。
- **恢复**：从之前的备份中恢复系统数据。
- **高级**：管理仓库、查看已下载镜像、导出系统日志以及配置系统级环境变量。
