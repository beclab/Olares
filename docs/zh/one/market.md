# 安装与更新应用 <Badge type="tip" text="10 min" />

Olares Market 是一个开放且无需许可的应用平台。你可以在这里找到官方 Olares 应用、第三方工具和 AI 服务。

本指南将带你完成最常见的应用管理任务，包括查找和安装应用、应用更新以及移除不再使用的应用。

## 学习目标

- 从 Olares Market 查找应用。
- 安装应用。本指南以 "Jellyfin" 为例。
- 将应用更新到最新版本。
- 卸载应用以释放系统资源。

## 安装应用

浏览 Market 目录并在设备上安装新软件。

1. 从 Dock 或 Launchpad 打开 Market。

    ![打开 Market](/images/one/open-market.png#bordered)

2. 通过分类浏览或使用左侧边栏的 **Search** 查找目标应用 "Jellyfin"。

    ![在 Market 中搜索](/images/one/search-app.png#bordered)

3. 点击 **Get**，然后点击 **Install**。

    ![打开 Market](/images/one/install-app.png#bordered)

    :::tip 进度指示器
    安装时间取决于应用大小和网络速度。**Install** 按钮会变为进度条，显示下载和安装状态。你可以在后台下载时继续使用 Olares。
    :::

4. 安装完成后，按钮变为 **Open**。点击即可立即启动应用。

    ![打开应用](/images/one/open-app.png#bordered)

## 更新应用

定期更新提供新功能、错误修复和安全补丁。通过安装更新，保持应用安全且功能丰富。

1. 从 Dock 或 Launchpad 打开 Market。
2. 在左侧边栏中点击 **Updates**。徽章数字表示可用更新的数量。

    ![更新应用](/images/one/update-app.png#bordered)

3. 选择你的更新方式：
    - 点击特定应用下方的 **Update** 单独更新该应用。
    - 点击 **Update all** 同时更新列表中的所有应用。

## 卸载应用

移除不再需要的应用，以节省存储空间。

1. 从 Dock 或 Launchpad 打开 Market。
2. 点击 **My Olares** 查看所有已安装的应用。
3. 找到目标应用 "Jellyfin"，点击 **Open** 按钮旁边的 <i class="material-symbols-outlined">keyboard_arrow_down</i>，然后点击 **Uninstall**。

    ![卸载应用](/images/one/uninstall-app.png#bordered)

4. 在 **Uninstall** 窗口中，点击 **Confirm**。
