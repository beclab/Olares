# Olares 文档

## Olares 是什么？

Olares 是一款开源个人云操作系统，让你能够在本地拥有并管理自己的数字资产。你无需再依赖公有云服务，而可以在 Olares 上本地部署强大的开源平替服务或应用，例如使用 Ollama 托管大语言模型、使用 ComfyUI 生成图像，以及使用 Vane（原 Perplexica）打造本地化、注重隐私的 AI 搜索与问答体验。你可以坐拥云计算的强大威力，同时将其完全置于自己掌控之下。
