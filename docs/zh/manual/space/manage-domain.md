# 管理域名成员

完成自定义域名设置并创建组织和 Olares ID 后，可以在 Olares Space 的**域名管理**中管理成员。

## 移除用户

只能移除尚未在你的域名下创建 Olares ID 的用户。

1. 在 Olares Space 中，进入**域名管理**，点击域名旁边的**查看**。
2. 找到要移除的用户名，点击**删除**。
3. 在弹出的对话框中点击**确认**。

## 下一步

- 用户收到 Olares ID 和密码后，可以[在 LarePass 中加入组织](../larepass/create-org-account.md#加入已有组织)。
- 如果成员将加入管理员现有的 Olares 实例，请参阅[管理团队](../olares/settings/manage-team.md)。
