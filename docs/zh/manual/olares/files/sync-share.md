# 同步与分享文件

LarePass 不仅能够确保你的文件内容在所有设备上保持一致且随时可访问，还能促进 Olares 服务器内的无缝协作。

本页面主要包含以下内容：
- **核心概念**：资料库、角色和权限。
- **资料库设置**：如何创建新的资料库以组织数字内容。
- **同步**：如何在资料库与本地电脑之间开启双向同步。
- **资料库管理**：如何与团队成员分享资料库并管理用户权限。
- **同步管理**：如何控制同步状态（暂停/取消同步）并处理同步冲突。

## 准备工作

### 理解同步与资料库

#### 同步 (Sync)

**文件** 应用中的同步功能类似于 iCloud 等云存储服务，你可以让最重要的信息保持最新，并在所有设备上随时可用。同步功能也让你能轻松地与 Olares 服务器内的其他成员分享文件。

#### 资料库 (Library)

资料库是组织、同步和分享数字内容的基本单位。它不仅仅是一个文件夹，而是一个通用的容器，旨在满足各种数据同步和分享需求：
* **多设备同步**：资料库确保你的数据在所有设备上保持一致。
* **实时协作**：与其他用户分享资料库，实现对同一资料库内数据的同时访问和编辑。
* **灵活管理**：创建多个资料库来组织不同类型的数据或用于不同的项目，让你能精细地控制同步和分享偏好。

### 角色与权限

此处描述的角色和权限专用于“文件”应用内的资料库文件分享和管理。这些与 Olares 的整体用户角色和系统级权限是区分开的。

| 操作 | 所有者 (Owner) | 成员 (Member) |
| :--- | :---: | :---: |
| 创建资料库 | ✅ | ✅ |
| 管理资料库权限 | ✅ | ❌ |
| 邀请其他成员 | ✅ | ❌ |
| 分享及重命名资料库 | ✅ | ❌ |
| 移除成员 | ✅ | ❌ |
| 删除资料库 | ✅ | ❌ |
| 退出资料库 | ❌ | ✅ |

权限级别：
* **只读 (Read-only)**：用户可以查看资料库内容，但无法修改。
* **读写 (Read-write)**：用户可以添加、删除和修改资料库内容。

### 前提条件

请确保已从[官网](https://www.olares.com/larepass)安装 LarePass 桌面客户端，并已使用你的 Olares ID 登录。

:::info
目前，本地文件同步功能适用于 Windows 和 Mac 用户。此文档将以 Mac 版本为例进行演示。
:::

## 创建资料库

每个用户都会自动获得一个个人资料库作为起点。要创建新的资料库：

1. 在 **同步 (Sync)** 目录右侧，点击 <i class="material-symbols-outlined">add_circle</i>。
2. 在 **新建资料库** 弹窗中，输入资料库名称，然后点击 **创建**。

## 开启同步

### 同步文件至本地电脑

资料库创建完成后，你可以将其映射到本地电脑以便访问。

1. 打开 Mac 上的 LarePass 桌面端。
2. 将鼠标悬停在目标资料库上，点击右侧出现的 <i class="material-symbols-outlined">more_horiz</i>，然后点击 **同步至本地**。
3. 指定本地目录，然后点击 **完成**。

    同步随即开始。完成后，该文件夹图标的左下角会出现一个绿色对勾，表示双向同步已激活。

### 同步本地文件至资料库

建立同步关系后，同步将自动进行。

:::info
如果你对资料库的权限是“只读”，则无法将本地文件夹的更改同步到资料库。你新添加和修改的文件将呈只读状态，并显示灰色的禁用图标 <i class="material-symbols-outlined">remove</i>。
:::

  * **上传**：将任何文件拖放到电脑上的本地同步文件夹中，它会自动上传到资料库。
  * **编辑**：从本地文件夹打开文件，编辑并保存。更改将自动同步到资料库。

## 管理资料库

你可以与团队成员分享资料库、设置权限，或删除不再需要的资料库。

:::tip
要在 Olares 中添加成员，请参阅[管理团队](https://www.google.com/search?q=../settings/manage-team.md)。
:::

### 分享资料库

你可以与 Olares 服务器内的其他成员分享资料库：

1. 将鼠标悬停在目标资料库上，点击右侧出现的 <i class="material-symbols-outlined">more_horiz</i>，然后点击 **分享**。
2. 在 **邀请用户** 区域，点击 <i class="material-symbols-outlined">add</i>，搜索并选择目标用户或群组，然后点击 **邀请**。
3. 在 **设置用户权限** 列表中，点击用户头像右侧的 <i class="material-symbols-outlined">chevron_forward</i>，为每位邀请用户分配特定的读写权限，然后点击 **提交**。
4. 点击 **确认**。

被邀请的用户将在其同步内容列表中看到该分享的资料库。要撤销分享权限，只需从分享窗口中移除该用户即可。

### 退出或删除资料库

如果你不想继续分享某个资料库，可以选择退出分享或将其删除。

  - **退出分享**：任何成员都可以退出已分享的资料库。当所有者退出时，该资料库将出现在其个人资料库列表中。
  - **删除**：只有所有者可以删除已分享的资料库。
    
    :::warning
    删除资料库是不可逆的。共享资料库中的所有文件将被永久删除。
    :::

1. 要退出资料库：

    a. 选择一个已分享的资料库，点击右侧出现的 <i class="material-symbols-outlined">more_horiz</i>，然后点击 **退出分享**。

    b. 点击 **确认**。

2. 要删除资料库：

    a. 选择一个已分享的资料库，点击右侧出现的 <i class="material-symbols-outlined">more_horiz</i>，然后点击 **删除**。

    b. 点击 **确认**。

## 管理同步

你可以通过以下方式管理同步状态。

### 定位本地文件夹

如果你想快速定位本地硬盘上的同步目录，将鼠标悬停在目标资料库或文件夹上，点击右侧出现的 <i class="material-symbols-outlined">more_horiz</i>，然后点击 **打开本地同步文件夹**。系统将直接打开该文件夹所在的本地位置。

### 取消同步

如果你不再需要同步某个文件夹，将鼠标悬停在该文件夹上，点击右侧出现的 <i class="material-symbols-outlined">more_horiz</i>，然后点击 **取消同步**。

### 暂停同步

如果你需要暂时停止数据传输，点击 **同步** 目录右侧的 <i class="material-symbols-outlined">pause_circle</i>。所有同步任务将被暂停。

## 处理同步冲突

在极少数情况下，如果多台设备同时编辑同一文件，LarePass 会自动处理冲突以防止数据丢失：

  * 最先完成的编辑将被保存到资料库。
  * 冲突版本将作为一个备份被创建，文件名包含编辑者的 Olares ID 和时间戳：`test.txt(SFConflict name 2024-04-17-12-12-12)`。
