# Olares One 文档

Olares One 是一款专为个人 AI 云打造的专属硬件设备。它将工作站级的强劲硬件与 Olares OS 系统结合，让你能够在本地流畅运行各类 AI 工具并安全存放数据。从此，你无需再将个人文件、隐私数据和提示词（Prompts）上传至云端服务器，真正实现对数字资产的完全掌控。

通过 Olares One，你可以轻松实现以下核心场景：

* **边缘 AI**：在本地运行开源 AI 模型，极速生成文本或图音视频内容，确保所有数据绝对“不出端”。
* **畅玩游戏**：借助 Steam Headless 应用，将设备瞬间化身高性能游戏主机，在 Olares One 或其他兼容设备上尽情游玩。
* **个人知识库**：让 AI 深度读取并理解你的私人文档，为你量身打造随叫随到的专属数字助手。
* **文件存储**：通过自带的 Files 应用，轻松实现个人文件的一站式集中管理、多端同步与安全分享。
* **私有工作空间**：一键在本地部署专属的办公与团队协作套件，打造高效、私密的数字办公环境。
* **私有媒体库**：搭配 Jellyfin 等应用搭建个人影视中心，随时随地串流播放你的海量电影与音乐收藏。
