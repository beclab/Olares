   :::tip Don't have the USB drive?
   Download the [Olares One ISO](https://cdn.olares.cn/one/olares-latest-amd64.iso), which is device-specific and different from the standard Olares ISO, and flash it to a USB drive (8 GB or larger) using a tool such as [Balena Etcher](https://etcher.balena.io/).
   :::
