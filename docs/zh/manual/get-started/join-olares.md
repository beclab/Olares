# 加入 Olares 集群

如果管理员邀请你加入现有的 Olares 集群，请按照本指南创建你的 Olares ID 并激活账号。整个过程均可远程完成，无需接触 Olares 设备。

## 步骤 1：创建 Olares ID

Olares ID 是你在 Olares 生态中的身份标识。激活账号前需要先完成创建。

你的 Olares ID 域名必须与 Olares 集群所使用的域名一致。创建前，请先向管理员确认所用域名。

具体步骤见[创建 Olares ID](./create-olares-id.md)。

## 步骤 2：激活账号

准备好 Olares ID 后，使用管理员提供的向导 URL 和一次性密码完成激活。此过程会通过 LarePass 将 Olares 集群与你的 Olares ID 关联。
