# 获取技术支持

如果你无法通过故障排查指南解决问题，可以联系 Olares 团队获取帮助。

为了更快地解决问题，请在联系支持团队前先收集并导出系统日志。

## 第二步：选择支持渠道

你可以通过设备上的 Ticket 应用提交请求，或者使用 Olares GitHub 仓库。

### 方式一：使用 Ticket 应用

你可以通过 Ticket 应用直接从 Olares 设备寻求帮助，在一个地方完成提问、报告问题和跟踪回复。

1. 打开应用市场，搜索 “Ticket”，并安装该应用。
2. 打开 Ticket 应用，屏幕上会显示登录二维码。
3. 在你的移动设备上，打开 LarePass 应用，进入**设置**标签页。
4. 点击右上角的扫描图标，扫描屏幕上的二维码，然后点击**确认**登录。
5. 提交工单：

   a. 点击左侧菜单或页面底部的**创建工单**。

   b. 描述你的问题，并上传导出的系统日志。

6. 查看问题进度：

   a. 在左侧边栏选择**所有工单**。

   b. 查看状态：待处理、处理中、已解决或已关闭。

   c. 确认状态后，点击工单查看详情。

### 方式二：使用 Olares GitHub 仓库

如果你无法访问 Olares 桌面，或希望公开报告问题，可以使用 GitHub。

1. 打开 [Olares GitHub 仓库](https://github.com/beclab/Olares)，选择以下方式之一：
    - 创建一个新的 **[Discussion](https://github.com/beclab/Olares/discussions/new?category=q-a)**（适合一般问题或需求帮助）。
    - 提交一个 **[Issue](https://github.com/beclab/Olares/issues/new)**（适合报告 Bug 或技术问题）。
2. 描述问题，并提供以下信息：
    - 复现问题的具体步骤
    - 错误信息或异常行为的描述
    - 使用环境的信息（操作系统、Olares 版本等）
    - 导出的系统日志文件
