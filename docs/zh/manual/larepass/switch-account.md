# 切换 Olares ID

如果你已在 LarePass 中添加多个 Olares ID，可以随时在移动端或桌面端切换使用。

## 移动端

1. 通过以下任一方式打开切换账户页面：

   - **从主界面进入**：点击左上角的个人头像。

      ![从主界面切换 Olares ID](/images/manual/larepass/switch-account-mobile1.png#bordered)

   - **在账户创建过程中**：点击右上角的账户图标。

      ![在账户创建过程中切换 Olares ID](/images/manual/larepass/switch-account-mobile2.png#bordered)

2. 在切换账户页面中，从列表里点击你想使用的账户。

   ![从列表中选择账户](/images/manual/larepass/switch-account-mobile.png#bordered)

## 桌面端

1. 点击左上角的个人头像，然后点击**切换账户**。
2. 在列表中点击你想使用的账户。
   ![在桌面端切换 Olares ID](/images/manual/larepass/switch-account-desktop.png#bordered)
