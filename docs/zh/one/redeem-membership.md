# 兑换 Olares Space 会员

根据你的购买渠道（如 Kickstarter 众筹活动），你的 Olares One 套装可能包含一张 Olares Space 会员卡。这张卡为你提供为期 2 年的免费 Basic 会员，可使用 Olares Space 提供的增强连接服务。

:::tip Olares One 是一次性购买
你的 Olares One 硬件和系统更新永久免费。

Olares Space 完全可选，你永远不会被锁定。如果你在免费期结束后选择不续订会员，你的 Olares One 使用不会受到任何影响。
:::

## 开始之前

Olares Space 是一个安全的连接平台，在扩展本地设备功能的同时不损害数据主权。Basic 计划包括：
* **云备份**：最多 50 GB 的集成云存储，用于关键数据冗余。免费替代方案是将数据备份到本地外置硬盘或你自己的 S3 兼容存储提供商。
* **远程访问（FRP）**：高速远程连接，每月 50 GB 流量，速度最高可达 50 Mbps。包括访问区域加速节点以获得更快的响应时间。如果超出每月配额，速度将被限制为 5 Mbps。免费替代方案是使用 LarePass VPN 或自行托管 FRP 服务器。

## 登录 Olares Space

1. 访问 https://space.olares.com。

    ![Olares Space 登录页面](/images/one/olares-space-login.png#bordered)

2. 在手机上打开 LarePass 应用，使用你的 Olares ID 登录。
3. 进入 **Settings** 选项卡，然后点击扫描图标。

    ![扫码登录](/images/one/scan-olares-space.png#bordered)

4. 扫描电脑屏幕上的二维码，点击 **Confirm**。

    现在你已经使用 Olares ID 登录了 Olares Space。

    ![登录 Olares Space](/images/one/login-olares-space.png#bordered)

## 兑换 Basic 计划

:::info
2 年有效期从你成功将邮箱关联到 Olares ID 的那一刻开始计算。
:::

要兑换 Basic 计划权益，你必须使用 Olares Space 仪表板将你的账户关联到与购买订单关联的邮箱地址。例如，与 Kickstarter 订单关联的邮箱。

1. 在 Olares Space 中，从左侧导航面板点击 **Usage & billing**。
2. 在 **Account email** 选项卡上，点击 **Email address**。

    ![Olares Space 账户邮箱](/images/one/account-email.png#bordered)

3. 在 **Add email** 窗口中，输入用于购买订单的邮箱地址。

    ![Olares Space 关联邮箱](/images/one/bind-email.png#bordered)

4. 点击 **Send verification code**。
5. 按照屏幕上的说明完成验证。

    :::tip
    如果你在收件箱中没有看到邮件，请尝试检查垃圾邮件或废品邮件文件夹。
    :::

    通知消息将确认邮箱已添加。你的账户立即升级为 Basic 计划，2 年有效期开始计算。

6. 点击右上角的头像。用户名旁边出现 **Basic** 标签即表示你的会员已激活。

    ![Olares Space Basic 计划](/images/one/basic-plan.png#bordered)

## 将会员转移到另一个 Olares ID

:::warning 有效期调整
将你的会员转移到新的 Olares ID 会使剩余有效期减少一个月。如果在同一日历月内多次转移，只扣除一次。
:::

:::warning 更改关联邮箱
Basic 会员严格绑定到用于购买订单的特定邮箱地址。如果你更改邮箱，当前账户将立即失去 Basic 会员资格。
:::

如果你想将 Basic 会员转移到新的 Olares ID：

1. 使用旧的 Olares ID 登录 Olares Space。
2. 进入 **Usage & billing** > **Account email**。
3. 点击 **Unbind** 从账户邮箱设置中移除购买订单邮箱。

    ![解绑邮箱](/images/one/unbind-button.png#bordered)

4. 点击 **Send verification code**。
5. 按照屏幕上的说明完成验证。

    通知消息将确认邮箱已移除。当前账户立即变为 Free 计划。

6. 使用新的 Olares ID 登录 Olares Space。
7. 使用[兑换 Basic 计划](#兑换-basic-计划)部分的步骤关联购买订单邮箱。这将立即将新的 Olares ID 升级为 Basic 计划。

## 常见问题

### 如果取消关联我的邮箱会发生什么？

你的邮箱关联到 Olares One 免费 Basic 计划。如果你取消关联：
- 当前账户立即降级为 Free 计划。
- 此账户上的所有 Basic 计划权益结束。
- 有效期继续倒计时。

### 如果取消了邮箱关联，我可以重新获得访问权限吗？

是的，你可以在到期之前重新关联同一邮箱，以恢复剩余有效期。在所有情况下，剩余有效期都会继续流逝，但是否扣除取决于你关联的账户：
- 如果你关联到另一个 Olares ID：该 ID 将获得权益，剩余有效期减少 1 个月。同一日历月内的多次转移只扣除一次。
- 如果你重新关联到同一账户：剩余有效期恢复，不扣除任何费用。它只会随时间流逝而减少。

### 如果切换到 Free 计划会发生什么？

如果你在 Olares One 免费 Basic 计划上手动更改为 Free 计划：
- 此账户上的所有 Basic 计划权益立即结束。
- 有效期继续倒计时。

你可以在有效期到期之前随时免费升级回当前账户的 Basic 计划：
1. 进入 **Usage & billing** > **Plan**。
2. 点击 **Change**。
3. 选择 **Basic** 并确认。你的账户将恢复为 Basic 计划。

### 2 年 Basic 会员到期后会发生什么？

当你的免费 Basic 计划到期时：
- 你的账户自动降级为 Free 计划。
- 所有 Basic 计划权益结束。
