# 在 LarePass 中管理账户

本指南介绍如何导入已有 Olares ID、切换账户，以及从当前设备中删除账户。

## 导入账户

在新设备或其他 LarePass 客户端上使用已有 Olares ID 时，可以通过 12 个助记词导入账户。

:::info 准备好助记词
导入 Olares ID 需要对应的 12 个助记词。如果尚未获取助记词，请先在已有该账户的设备上[备份助记词](back-up-mnemonics.md)。
:::

### 移动端

1. 打开 LarePass 应用。
2. 点击个人头像。
3. 在**切换账户**页面底部，点击**添加新账户**。
4. 点击**导入账户**。
5. 输入 Olares ID 对应的 12 个助记词。

   ![在移动端导入账户](/images/manual/larepass/import-account-mobile.png#bordered)

### 桌面端

1. 打开 LarePass 桌面客户端。
2. 点击个人头像。
3. 点击**切换账户**。
4. 点击底部的**添加新账户**。
5. 输入 Olares ID 对应的 12 个助记词。

   ![在桌面端导入账户](/images/manual/larepass/import-account-desktop.png#bordered)

## 切换账户

如果已在 LarePass 中添加多个 Olares ID，可以随时切换使用。

### 移动端

1. 通过以下任一方式打开**切换账户**页面：

   - **从主界面进入**：点击左上角的个人头像。

      ![从主界面打开切换账户页面](/images/manual/larepass/switch-account-mobile1.png#bordered)

   - **在账户创建过程中**：点击右上角的账户图标。

      ![在账户创建过程中打开切换账户页面](/images/manual/larepass/switch-account-mobile3.png#bordered)

2. 点击要使用的账户。

   ![选择账户](/images/manual/larepass/switch-account-mobile4.png#bordered)

### 桌面端

1. 点击左上角的个人头像，然后点击**切换账户**。
2. 点击要使用的账户。

   ![在桌面端切换账户](/images/manual/larepass/switch-account-desktop1.png#bordered)

## 删除账户

如果不想继续在当前设备上保留某个账户，可以将其从 LarePass 中删除。

:::warning 备份助记词
删除账户前，必须先[备份助记词](back-up-mnemonics.md)。未备份助记词的账户无法删除。
:::

### 移动端

1. 打开 LarePass 应用。
2. 点击个人头像。
3. 打开**切换账户**页面。
4. 点击右上角的 <i class="material-symbols-outlined">checklist</i>。
5. 选择要删除的账户。
6. 点击右上角的 <i class="material-symbols-outlined">delete</i>，然后确认删除。

   ![在移动端删除账户](/images/manual/larepass/delete-account-mobile.png#bordered)

### 桌面端

1. 点击左上角的个人头像，然后点击**设置**。
2. 下拉至**账户**，点击**管理账户**。
3. 点击**删除账户**删除当前账户，然后确认删除。

   ![在桌面端删除账户](/images/manual/larepass/delete-account-desktop.png#bordered)
