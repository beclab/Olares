# Olares Space overview

Olares Space provides essential infrastructure services for your Olares environment, including data backup, custom domain configuration, reverse proxy, and shared GPU resources.

Your Olares Space account is associated with your Olares IDs or DIDs. You can log in to Olares Space by scanning a QR code using LarePass.

This section walks you through using Olares Space:

- [Manage domain members](manage-domain.md)
