# Olares 文档

## Olares 是什么？

Olares 是一款可以用自然语言操作的开源个人云操作系统，让你在自己的硬件上运行 AI Agent 和大模型。

你可以通过本文档完成 Olares 的安装与激活，管理应用、文件、身份和访问方式，并借助 CLI、Agent Skills 和真实场景示例进一步扩展 Olares。
