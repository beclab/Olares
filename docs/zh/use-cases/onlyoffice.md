# OnlyOffice

OnlyOffice 是 Olares 上用于文档编辑与协作的共享应用。

## 从 v2 迁移到新架构

Olares 1.12.6 引入了新的共享应用架构。迁移 OnlyOffice 的步骤如下：

1. 备份 v2 应用数据。
2. 卸载 v2 版 OnlyOffice。
3. 安装新版 OnlyOffice。
4. 恢复你的文档和应用设置。

通用迁移流程请参阅[从 v2 迁移到新架构](../manual/olares/market/shared-apps.md#option-3-back-up-uninstall-v2-install-the-new-app-then-restore-data)。
