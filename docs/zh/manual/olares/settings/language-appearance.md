# 设置语言和外观

个性化 Olares 使用体验十分简单。你可以设置首选语言，在浅色和深色主题之间切换，并为桌面和登录屏幕选择自定义壁纸。

## 系统语言

Olares 当前支持以下语言：
- 英文
- 中文（简体）

更改显示语言的方法：

1. 打开**设置**，在左侧边栏选择**外观**。
2. 点击下拉菜单，选择语言。

## 主题

在系统应用中切换浅色和深色主题：

1. 打开**设置**，在左侧边栏选择**外观**。
2. 在**主题**面板中，选择**浅色**或**深色**。

更改会立即生效。

## 壁纸

自定义桌面或登录界面的背景：

1. 从应用坞打开**设置**。
2. 在左侧边栏选择**外观**。
3. 在**壁纸** 面板中，选择壁纸的应用位置：**桌面背景**或**登录背景**。
4. 选择壁纸：
   - 内置壁纸：浏览内置图库的图片并选择。
   - 已上传图片：从上传过的壁纸中选择。
5. 如需上传自定义壁纸，点击**上传图片**，选择电脑中的图片文件，然后点击**打开**。
6. （可选）使用**上传图片**旁边的下拉列表调整壁纸显示方式：
   - **拉伸**
   - **填充**
   - **平铺**
7. （可选）管理已上传的壁纸：
   - 在**已上传图片** 区域，将鼠标悬停在图片上，点击右上角的 <i class="material-symbols-outlined">close</i> 图标即可删除。
