# 访问 Olares 主机终端

某些开发和运维任务需要在 Olares 主机上运行命令，例如检查磁盘、验证主机状态或更新主机级配置。由于 Olares 主机通常以无显示器、无键盘的方式部署，因此终端访问主要通过远程方式进行。

你可以通过以下两种方式访问主机终端：

1. **Secure Shell（SSH）**：标准的远程管理。
2. **控制面板终端**：通过 Olares 网页端直接获取 root 权限。

## 通过 SSH 访问

SSH 是从远程开发设备操作 Olares 主机的标准协议，可在网络上建立安全的终端会话。

### 前提条件

连接之前，请确保满足以下条件：

- 与 Olares 主机建立网络连接  
  - 在大多数情况下，你的电脑和 Olares 主机位于同一局域网内。  
  - 如果需要从局域网外部连接，请先配置 VPN 访问。详见[通过 LarePass VPN 连接](#通过-larepass-vpn-连接)。
- Olares 主机 IP 地址，通常为 `192.168.x.x`。
- 有效的登录凭证。

:::info
在 Olares Zero 设备上，默认的 SSH 用户名和密码均为 `olares`。
:::

### 通过局域网连接

1. 打开计算机上的终端。
2. 使用以下格式运行 SSH 命令：

   ```bash
   ssh <username>@<host_ip_address>
   ```

   示例：
   ```bash
   ssh olares@192.168.31.155
   ```
3. 根据提示输入密码。

### 通过 LarePass VPN 连接

如果你的电脑与 Olares 主机不在同一局域网，请启用 LarePass VPN。

1. 在 Olares 中，前往**设置** > **VPN**，然后启用**允许通过 VPN 进行 SSH 连接**。
2. 打开 LarePass 桌面客户端，点击左上角头像打开用户菜单。
3. 打开**专用网络连接**开关。
4. 打开计算机上的终端。
5. 使用以下格式运行 SSH 命令：

    ```bash
    ssh <username>@<host_ip_address>
    ```
    
    示例：
    ```bash
    ssh olares@192.168.31.155
    ```
6. 根据提示输入密码。

## 通过控制面板访问

如果你能登录 Olares 网页端，可以直接通过 Olares 控制面板打开主机终端。

1. 打开**控制面板**。
2. 在左侧边栏的终端部分，点击 **Olares**。

:::info Root 权限
通过控制面板打开的终端默认以 `root` 身份运行。

请勿在命令前使用 `sudo`。例如，直接运行 `apt update`，而不是 `sudo apt update`。
:::

## 了解 root 角色

:::warning 请谨慎操作
root 权限允许修改系统关键文件和配置。不当的操作可能导致系统不稳定或数据丢失。
:::

`root` 用户是系统中超级管理员，在 Olares 主机上拥有完全权限。当你通过控制面板访问终端时，系统会自动以 `root` 身份执行命令。这种设计简化了管理任务，默认可以运行高权限命令。

### 确认当前用户角色

你可以通过以下命令确认当前终端会话所使用的用户角色：

```bash
whoami
```
你也可以通过终端提示符判断权限级别：

- 以`#`结尾的提示符表示当前为 root 会话。

    示例：`root@host:~#`

- 以`$`结尾的提示符表示当前为普通用户会话。

    示例：`olares@host:~$`
