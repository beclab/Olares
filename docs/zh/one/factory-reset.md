# 通过 LarePass 恢复出厂设置 <Badge type="tip" text="10 min" />

:::warning
本页面内容经 AI 翻译生成，仅供参考。具体细节请以[英文原文](../../one/factory-reset.md)为准。
:::

如果你已经激活了 Olares One，并希望将其恢复到出厂状态，可以在 LarePass 中执行恢复出厂设置操作。

:::warning 数据丢失
此操作将永久删除设备上的所有账户、设置和数据。该操作无法撤销。
:::

## 前提条件

**硬件**<br>
- Olares One 已开机。
- 你的手机与 Olares One 处于同一网络。
- 你以管理员身份登录 LarePass。

## 将 Olares One 恢复出厂设置

1. 在手机上打开 LarePass，进入 **设置**。
2. 在 **我的 Olares** 卡片中，点击 **系统** 打开 **Olares 管理** 页面。
3. 点击 **恢复出厂设置**。
4. 仔细阅读风险提示，然后点击 **恢复出厂设置** 继续。
    ![查看风险提示](/images/manual/larepass/review-risk-prompt.png#bordered)

5. 输入你的本地 LarePass 锁屏密码，然后点击 **确认**。
    ![输入本地解锁密码](/images/manual/larepass/enter-password-to-uninstall.png#bordered)
   
   如果你尚未设置本地密码，系统会提示你先设置一个。
6. 等待恢复过程完成。

恢复完成后，Olares One 将重新启动并进入初始设置阶段。你会被引导至激活流程，可以在其中扫描本地网络以重新激活 Olares。

详细说明请参阅[首次启动](first-boot.md)。
