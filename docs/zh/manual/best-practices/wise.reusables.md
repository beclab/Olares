将内容收集到 Wise 后，可以使用 Olares 的聚合搜索功能查找特定的主题或条目。

![Search in Wise CN](/images/zh/manual/tutorials/wise-search-cn.png#bordered)

::: info 支持的搜索格式
目前仅支持搜索文档类内容（PDF、网页文章和 ePUB）。其他格式将在未来支持。
:::

1. 点击 Dock 中的 <i class="material-symbols-outlined">search</i> 图标，打开搜索窗口。
2. 将搜索范围设置为 Wise，然后输入关键词开始搜索。
