# 了解桌面

桌面应用是用户与 Olares 系统交互的主要入口。它提供直观高效的方式来管理和使用系统内置应用以及用户安装的应用。

## 桌面基础

![桌面](/images/zh/manual/olares/desktop.png#bordered)

### 应用坞

应用坞是位于屏幕左侧的应用程序快速启动栏。你可以通过应用坞打开常用应用，访问桌面的主要功能。

### 启动台

启动台显示所有已安装的应用程序。点击应用坞中的启动台图标即可打开。

### 应用窗口

默认情况下，应用以**窗口模式**打开，作为桌面的内嵌页面运行。你可以像在电脑上一样管理窗口：

- 拖动标题栏移动窗口。
- 拖动窗口边缘调整大小。
- 最小化、最大化或关闭窗口。
- 点击 <i class="material-symbols-outlined">open_in_new</i> 图标，在新的浏览器标签页中打开应用。

:::tip 提示
部分应用仅支持以浏览器标签页方式打开。
:::

### 搜索与通知

- **搜索**：快速启动应用，并在 Olares 中查找支持的内容。
- **通知**：点击通知图标查看系统和应用通知。

## 使用启动台

通过启动台，你可以：

- 查看所有已安装的应用。
- 点击应用图标打开应用。
- 拖动图标，调整应用在启动台中的排列顺序。
- 将图标拖动到程序坞以便快速访问。

### 卸载应用

1. 长按应用图标进入编辑模式。
2. 如果应用图标左上角出现 <i class="material-symbols-outlined">close_small</i> 图标，点击即可卸载该应用。

:::tip 注意
系统内置应用（如文件管理器、应用市场、个人主页）无法卸载。
:::

### 应用坞与启动台

- **应用坞：** 屏幕侧边的快速启动栏，用于固定常用应用。
- **启动台：** 点击应用坞中的启动台图标可打开，展示所有已安装应用。

## 使用全局搜索

你可以通过以下任一方式快速打开 Olares 的全局搜索：

- 按下快捷键 `Shift + Space`
- 点击应用坞中的<i class="material-symbols-outlined">search</i> 图标

全局搜索可以查找应用、文件以及受支持的内容。

| 支持的搜索对象 | 支持的搜索操作 |
|:--|:--|
| 应用（内置与已安装） | 按应用名称搜索，并直接打开应用 |
| 启用了全文搜索的目录 | 按文件名或文件内容搜索并访问目录中的文件 |
| 文件管理器中的其他目录 | 按文件名搜索 |
| 团队共享文件 | 按文件名搜索与你共享的文件 |
| 文件管理器中的同步磁盘 | 按文件名搜索目录中的文件 |
| Wise 阅读器 | 按名称搜索 RSS 订阅源、网页和 PDF 等内容 |
| **更多内容** |  |
| 设置  | 调整 Olares 外观设置 |

### 配置文件搜索规则

默认情况下，全局搜索**只能通过文件名**搜索文件夹。  

如果你希望提升搜索效率，或者按**文件内容**进行搜索，请前往**设置** > **搜索** > **文件搜索**来配置全文搜索和排除规则。

详情请见[配置文件搜索](/zh/manual/olares/settings/search.md)。
