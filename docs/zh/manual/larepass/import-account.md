# 导入账户

你可以使用 12 个助记词将已有的 Olares ID 导入到 LarePass，从而在新设备或其他 LarePass 客户端上访问 Olares 服务。

:::tip 备份助记词
请确保已[备份助记词](back-up-mnemonics.md)，否则无法完成导入。
:::

<Tabs>
<template #iOS-&-Android>

1. 打开 LarePass 应用。
2. 点击头像。
3. 在**切换账户**页面底部，点击**添加新账户**。
4. 点击**导入账户**。
5. 输入 Olares ID 对应的 12 个助记词。

</template>
<template #macOS-&-Windows>

1. 打开 LarePass 桌面客户端。
2. 点击头像。
3. 点击**切换账户**。
4. 点击底部的**添加新账户**。
5. 输入 Olares ID 对应的 12 个助记词。

</template>
</Tabs>
