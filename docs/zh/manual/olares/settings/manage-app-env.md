# 管理应用环境变量

应用环境变量用于定义应用在容器中的运行方式，包括应用配置、服务连接以及其他运行时设置。

要查看或修改应用的环境变量：

1. 前往**设置** > **应用**。
2. 选择目标应用。
3. 在**环境变量**部分，点击**管理环境变量**。

![应用环境变量](/images/zh/manual/olares/manage-app-env1.png#bordered){width=90%}

## 应用环境变量类型

| 环境变量类型 | 说明 | 编辑权限 |
|:--|:--|:--|
| **引用的系统环境变量** | 应用所引用的系统环境变量，例如<br> `OLARES_USER_USERNAME`。<br>如果应用使用了系统环境变量，就会<br>显示在此处。 | **只读**。如需修改，前往**设置** > **高级** > **系统环境变量**进行修改。 |
| **应用专属环境变量**  | 当前应用的专属环境变量。 | **可编辑**。你可以直接在此页面修改其值，无需在控制面板编辑 YAML 文件。 |
